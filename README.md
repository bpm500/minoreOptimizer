## 📦 Установка и запуск

### Вариант 1: Запуск из исходников (Python)
```
pip install PyQt6
python optimizer.py
```

### Вариант 2: Сборка в EXE
1. Убедитесь, что установлен Python 3.10+
2. Положите ваш `icon.ico` и `icon.png` в папку рядом с `optimizer.py`
3. Запустите `build.bat` от имени администратора
4. Готовый EXE будет в папке `dist\`

## 🛡 Почему антивирус может срабатывать?

EXE-файлы, собранные PyInstaller, иногда ложно срабатывают на VirusTotal.  
Чтобы минимизировать это:

1. **Подпишите EXE** цифровой подписью (Code Signing Certificate)
   - Купить на: Sectigo, DigiCert, Comodo (~$80-200/год)
   - Команда подписи: `signtool sign /tr http://timestamp.digicert.com /td sha256 /fd sha256 /a minoreOptimizer.exe`

2. **Добавьте в исключения** Windows Defender / антивируса

3. **UPX не используйте** — он увеличивает подозрения

4. **MIT License** уже прописана в коде (первые строки файла)

5. Загрузите на **GitHub** с описанием — это повышает доверие

```

├── requirements.txt      # Зависимости

    └── MSIAfterburner.exe  # Положите сюда (опционально)
```
